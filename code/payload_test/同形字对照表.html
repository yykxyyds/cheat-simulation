六、隐形字符

下列字符在浏览器中完全不可见（零宽），但存在码点，可嵌入文本污染 tokenizer 而不改变渲染。

字符Code 名称 特殊效果
(零宽)U+200B 零宽空格（ZWSP） 允许换行，切开 token
(零宽)U+200C 零宽不连词（ZWNJ） 阻止字符连写
(零宽)U+200D 零宽连词（ZWJ） 强制字符连写
(零宽)U+2060 词连接符（WORD JOINER） 禁止此处换行
(零宽)U+FEFF 字节序标记（BOM / ZWNBSP） 文件头标识，正文通常不可见
(零宽)U+00AD 软连字符（SOFT HYPHEN） 换行处可能显示为连字符 -
(零宽)U+034F 组合字素连接符（CGJ） 影响字素聚类，不被 normalize 剥离
(零宽)U+200B+U+034F ZWSP + CGJ 组合 ZWSP 破坏 tokenizer + CGJ 影响字素聚类，不被 normalize 剥离
(零宽)ZWSP+ZWNJ 组合 零宽二进制编码（Zero-Width Binary） 用 ZWSP/ZWNJ 序列编码二进制信息，隐蔽传输
(零宽)U+2061 函数应用符（FUNCTION APPLICATION） 数学隐式函数，零宽
(零宽)U+2062 隐形乘号（INVISIBLE TIMES） 数学隐式乘法，零宽
(零宽)U+2064 隐形加号（INVISIBLE PLUS） 数学隐式加法，零宽
(零宽)U+2063 不可见分隔符（INVISIBLE SEPARATOR） 零宽分隔，不被 normalize 剥离
(零宽)U+200E 从左到右标记（LRM） 隐式强制 LTR 方向
(零宽)U+200F 从右到左标记（RLM） 隐式强制 RTL 方向
(零宽)U+202A 从左到右嵌入（LRE） 嵌入 LTR 方向层
(零宽)U+202B 从右到左嵌入（RLE） 嵌入 RTL 方向层

共 22 个大写字母 + 16 个小写字母 + 1 个数字 + 4 个特殊符号存在视觉同形；8 个空格变体 + 26 个隐形字符可替代普通空格或嵌入文本而不被察觉。
已纳入适用实验：大写 14 个希腊 + 多个西里尔；小写 7 个西里尔（a/c/e/o/p/x/y）。
最具新增实验价值的候选：小写 d→ԁ, q→ԛ, t→т, w→ԝ；大写 Q→Ԛ, W→Ԝ。
