## 三、使用方式

```bash
# 切换模型运行仿真实验
python pentest_agent_poison_sim.py \
  --scenario Labyrinth_Linguist \
  --experiments all \
  --model "deepseek-ai/deepseek-v4-pro"

# 多个模型对比
python pentest_agent_poison_sim.py --model "meta/llama-4-maverick-17b-128e-instruct" ...
python pentest_agent_poison_sim.py --model "qwen/qwen3.5-397b-a17b" ...
```

> **注意**：部分大模型（如 `mistral-large-3-675b`、`nemotron-ultra-253b`）推理速度较慢、费用较高，大规模实验前建议先单轮验证。
